URL Shortener Microservice

Pass a /new/URL as a parameter and receive a shortened URL as a JSON

When visiting the shortened URL, it will be redirected to the original link

Example creation usuage:

http://localhost:3000/new/https://www.google.com

http://localhost:3000/new/http://foo.com:8080

Example creation output

{"original_url":"https://www.google.com", "short_url":"http://localhost:3000/2"}

Usage:

http://localhost:3000/2

Will redirect to:

https://www.google.com
